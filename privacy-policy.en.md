# Privacy Policy

Last updated: February 2026

This Privacy Policy describes how the ThesisAudit application ("the App") collects, uses, and protects user information.

## 1. Information We Collect

We collect the following information:

- **Email address** — used for user registration, authentication, and account access.
- **Telegram account data** — if you choose to authenticate via Telegram, we receive basic public information provided by Telegram.
- **User-submitted content** — documents, audio recordings, and other materials that you voluntarily upload to the App for analysis.

We do not collect personal data unrelated to the functionality of the App.

## 2. How We Use Information

Collected information is used solely to:

- Authenticate users and manage accounts
- Provide access to application features
- Process user-submitted content
- Improve the stability and quality of the App

We do not use user data for advertising or marketing purposes.

## 3. Data Storage and Processing

User data is processed and stored on secure servers.  
We apply reasonable technical and organizational measures to protect data from unauthorized access, loss, or misuse.

## 4. Data Transmission Security

All data transmitted between the App and the server is sent via secure, encrypted connections using industry-standard security practices.

## 5. Data Sharing

We do not sell, rent, or share user data with third parties, except when required by law.

## 6. Data Retention

User data is stored only for as long as necessary to provide the App’s functionality or as required by applicable laws.

## 7. User Rights

Users have the right to request access to or deletion of their personal data by contacting us.

## 8. Children’s Privacy

The App is not intended for use by children under the age of 13.  
We do not knowingly collect personal data from children.

## 9. Changes to This Policy

We may update this Privacy Policy from time to time.  
Any changes will be published on this page.
